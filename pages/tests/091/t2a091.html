Question 1 - Word Problem - (10 marks):

Design and code a class named Stock that holds information about a stock item in a store.  Upon instantiation, a Stock object may receive nothing or may receive three pieces of information: an integer identifying the item, a double-precision, floating point value holding the price of the item and the address of a C-style null-terminated string describing the item.  A valid identifier is a positive integer.  A valid price is a positive value.  The description may be of any length.  If the object does not receive any data or the data provided is invalid in any way, the object adopts a safe empty state. 

Store your class declaration in a header file named Stock.h and your function definitions in an implementation file named Stock.cpp.  Include all of the prototypes necessary to ensure proper compilation and execution of your coding.

Since your design must accommodate a description of any length, your design should include a deep copy constructor, a deep assignment operator and a destructor.

Your design also includes a member operator and overloads the insertion operator:

  • Stock& operator+=(double raise) - a member operator that receives a double value and increases the price of the item by that value if the object is not empty.  The operator returns a reference to the current object.
  • the overloaded insertion operator outputs a stock item's information in the following form:
    • the item number padded with 0's in a field of eight(6)
    • the price of the item to 2 decimal places in a field of ten(10)
    • the description of the item

    The pieces of information are separated by a single space.  Design your operator so that it allows cascading.  If the object is empty, your operator outputs "empty object" in the description field as shown in the example below.

Question 2 - Walkthrough - (10 marks):

What is the exact output of the following program?  Show your rough work to avoid deductions. 

 #include <iostream>
 using namespace std;

 class Bread {
     double weight;
     int slices;
   public:
     Bread() {
         weight = 0.0;
         slices = 0;
         cout << '#';
     }
     Bread(double w, int s) {
         weight = w;
         slices = s;
         cout << '%';
     }
     Bread(const Bread& b) {
         weight = b.weight;
         slices = b.slices;
         cout << '^';
     }
     void operator--() {
         weight -= weight / slices;
         slices--;
     }
     void operator--(int) {
         weight -= 2 * weight / slices;
         slices -= 2;
     }
     ~Bread() {
         display();
         cout << '$' << endl;
     }
     void display() const {
         cout << weight << ',' << slices;
     }
 };

 class RaisinBread : public Bread {
     int raisins;
     int r_per_slice;
   public:
     RaisinBread() {
         raisins = 0;
         r_per_slice = 1;
         cout << '@';
     }
     RaisinBread(double w, int s, int r) : Bread(w, s) {
         raisins = r;
         r_per_slice = r / s;
         cout << '*';
     }
     RaisinBread(const RaisinBread& r) : Bread(r) {
         raisins = r.raisins;
         r_per_slice = r.r_per_slice;
         cout << '+';
     }
     void operator--() {
         Bread::operator--();  // calls base class operator
         raisins -= r_per_slice;
     }
     void operator--(int) {
         Bread::operator--(2);  // calls base class operator
         raisins -= r_per_slice * 2;
     }
     ~RaisinBread() {
         display();
         cout << '~' << endl;
     }
     void display() const {
         Bread::display();
         cout << ',' << raisins;
     }
 };

 int main() {
     RaisinBread rb(100.0, 10, 100), rd;
     Bread b(10,2);
     cout << "\n-----------" << endl; // 11 dashes

     rb.display();
     cout << endl;
     b.display();
     cout << endl;

     cout << "-----------" << endl;   // 11 dashes
     RaisinBread rc = rb;
     --rb;
     --b;
     --rb;
     cout << endl;
     rd.display();
     cout << endl;

     return 0;
