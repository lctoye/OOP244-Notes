Question 1 - Word Problem - (10 marks):

A cash card is a new type of plastic card, like a credit or debit card, that holds the equivalent of cash on an integrated chip, which is imbedded within the card itself.  It is similar to a gift card, but is issued by a bank or financial institution, and can be used at any store that has a debit or credit card machine. 

Design and code a class named CashCard that holds information about such a card.  Place your class declaration in a header file named CashCard.h and your function definitions in an implementation file named CashCard.cpp.  Include in your coding all of the statements necessary for your code to compile under a standard C++ compiler and to run successfully.

Upon instantiation, a CashCard object may receive no parameters or may receive a C-style, null terminated string holding the identifying number of the card and an int holding the PIN for the card.  The identifying number must consist of digits and must not contain more than 6 digits.  The PIN must be a whole number greater than 0.  If the data received by your object does not satisfy both of these conditions, your object sets itself to a safe empty state.  Upon instantiation, the object holds a cash balance of $0.00. 

Your class includes the following member functions:

  • int deposit(double amount) - a modifier that receives the amount to be deposited.  If this amount is a positive value, your function adds the amount to the cash already on the card and returns true.  Otherwise, your function does nothing and returns false. 
  • int withdraw(double amount, int pin) - a modifier that receives the amount to be withdrawn and a PIN.  If your object has enough cash to support this withdrawal and the same PIN as the PIN received, your function withdraws the requested amount and returns true.  Otherwise, your function does nothing and returns false. 
  • void display() const - a query that inserts information stored in your object to the standard output stream.  This function displays the identifying number right-justified in a field of 6 with 0 padding followed by the amount of cash on the card right-justified in a field of 6, showing 2 decimal places.  If your object is in a safe empty state, this function displays "empty card". 

As an example, consider the following program that uses your class 

 #include "CashCard.h"

 int main() {
     CashCard card("12341", 1234), x; 

     card.display();
     card.deposit(45.00);
     card.display();
     card.withdraw(120.00, 1234);
     card.display();
     card.withdraw(20.00, 1234);
     card.display();
     x.display();

     return 0;
 }

This program produces the following output 

 012341  0.00
 012341 45.00
 012341 45.00
 012341 25.00 
 empty card



Question 2 - Walkthrough - (10 marks):

What is the exact output of the following program?  Show your rough work to avoid deductions. 

 #include <iostream>
 using namespace std;
 #define MCHAR   4
 #define NTOKENS 4

 class Token {
     char str[MCHAR + 1];
   public:
     Token() {
         cout << '*';
         strcpy(str, "?");
     }
     ~Token() {
         display();
         cout << ',';
     }
     int extract(const char* in, int& i) {
         int rc = 1, j = 0, accept = 1;
         while (in[i] == ' ') i++;
         while (accept && j < MCHAR) {
             if (in[i] == '\0') {
                 str[j] = '\0';
                 rc = 0;
                 accept = 0;
             }
             else if (in[i] == ' ') {
                 str[j] = '\0';
                 accept = 0;
             }
             else
                 str[j++] = in[i];
             cout << i << ',' << j << '|';
             i++;
         }
         cout << endl;
         return rc;
     }
     void display() const { cout << str; }
 };

 int main() {
     Token token[NTOKENS];
     int rc, i = 0, noTokens = 0;
     char str[] = "244 is tuf";

     cout << '\n';
     cout << "----------------" << endl; // 16 -s
     do {
         rc = token[noTokens].extract(str, i);
         noTokens++;
     } while (rc != 0);
     for (int i = 0; i < noTokens; i++) {
         cout << i << ' ';
         token[i].display();
         cout << endl;
     }
     cout << "----------------" << endl; // 16 -s 

     return 0;
 }
