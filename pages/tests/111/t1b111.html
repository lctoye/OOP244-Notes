Question 1 - Word Problem (10 marks)

Design and code a class named CreditCard that keeps track of the credit available on a credit card.  Upon instantiation, a CreditCard object may receive either no arguments or three arguments.  The three arguments are the card number, the PIN number, and the credit limit in that order.  If there are no arguments or if the card number or credit limit are not positive values, the object assumes a safe empty state.  A newly created object assumes that the full amount of the credit limit is available. 

Your design includes two constructors - the no-argument constructor and the three-argument constructor - and the following methods: 

  • void display() const - a query that displays on standard output (as shown in the example below):
    • the last four digits of the card number in a field width of 8 padded with '*'s 
    • the amount of credit available on the card to 2 decimal places in a field width of 8
    If the object is in a safe empty state, this method displays "Card is inactive"

  • bool charge(int pin, double amount) - a modifier that accepts a charge against the credit card.  If the pin number is valid and there is sufficient credit available to cover the amount charged, this method reduces the credit available by the amount charged and returns true; otherwise, this method returns false

  • void settle(double payment) - a modifier that pays down the amount charged on the card by the payment received and adjusts the credit available

Place your class declaration in a header file named CreditCard.h and your function definitions in an implementation file named CreditCard.cpp.  Include all coding necessary to compile and run successfully under a standard C++ compiler.  The following program that uses your design produces the output shown on the right 

 #include <iostream>

Question 2 - Walkthrough (10 marks)

What is the exact output of the following program?  Show your rough work to avoid deductions. 

 #include <iostream>
 using namespace std;

 class Peanut {
     int size;
   public:
     Peanut();
     Peanut(int s);
     void bite();
     ~Peanut();
 };

 class Jar {
     int no;
     Peanut peanut[4];
   public:
     Jar(int n);
     void takeBite(int i);
     void swallow(int i);
     ~Jar();
 };

 Peanut::Peanut() {
     size = 0;
     cout << "p0" << size << endl;
 }
 Peanut::Peanut(int s) {
     size = s;
     cout << "ps" << size << endl;
 }
 void Peanut::bite() {
     size = size / 2;
     cout << "pf" << size << endl;
 }
 Peanut::~Peanut() {
     cout << "p~" << size << endl;
 }

 Jar::Jar(int n) {
     Peanut local(4);
     for (int i = 0; i < n; i++)
         peanut[i] = local;
     no = n;
     cout << "jc" << no << endl;
 }
 void Jar::takeBite(int i) {
     if (i >= 0 && i < no)
         peanut[i].bite();
 }
 void Jar::swallow(int i) {
     Peanut local;
     if (i >= 0 && i < no)
         peanut[i] = local;
     if (i == no - 1)
         no--;
 }
 Jar::~Jar() {
     cout << "j~" << no << endl;
 }

 int main() {
     Jar jar(3);
     jar.takeBite(1);
     jar.swallow(2);
     jar.takeBite(1);
 }
